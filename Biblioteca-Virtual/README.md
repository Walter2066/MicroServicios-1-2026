# Biblioteca Virtual

Aplicación web desarrollada con **Node.js, Express, MongoDB y EJS** que permite gestionar un catálogo de libros mediante operaciones **CRUD (Create, Read, Update, Delete)**.

El sistema permite registrar libros, visualizar el catálogo, editar información existente y eliminar registros de la base de datos.

---

# 🗄️ Base de datos

La aplicación utiliza **MongoDB** con la base de datos:

```
db_biblioteca_v
```

---

# 🚀 Despliegue del proyecto 

Para ejecutar el proyecto en cualquier entorno se recomienda seguir estos pasos:

1. Clonar el repositorio:

2. Entrar al directorio del proyecto:

```
cd Biblioteca-Virtual
```

3. Instalar todas las dependencias necesarias:

```
npm install
```

4. Verificar que MongoDB esté ejecutándose en el sistema.

5. Iniciar la aplicación:

```
npm run dev
```

6. Abrir el navegador en:

```
http://localhost:3000
```

---

# 📌 Recomendaciones

* Asegurarse de tener **Node.js 24.14.0** y **MongoDB** instalados en el sistema.
